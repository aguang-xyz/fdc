◆ canonical()

fds fdc::canonical ( const fds &  F)

Given a set of functional dependencies \( F \), calculate \( G^+ = F^+ \) where

  • \( G \) is non-redundant.
  • For every \( X \to Y \in G \), \( |Y| = 1 \) and there is no such a \( X^{'} \subset X \) where \( X^{'} \to Y \in G \).

Time complexity: \( O(|F|^2) \).

See also: 5.2 Redundancy Tests in Beeri and Bernstein (1979, p. 47)

FA set of cuntional dependencies.

◆ is_canonical()

bool fdc::is_canonical ( const fds &  F)

Given a set of functional dependencies \( F \), determine if \( F \) is canonical.

The definition of canonical:

  • \( F \) is non-redundant.
  • For every \( X \to Y \in F \), \( |Y| = 1\) and there is no such a \( X^{'} \subset X \) where \( X^{'} \to Y \in F \).

Time complexity: \( O(|F|^2) \).

See also: 5.2 Redundancy Tests in Beeri and Bernstein (1979, p. 47)

Parameters
FA set of cuntional dependencies.

◆ is_redundant()

bool fdc::is_redundant ( const fds &  F)

Check if a set of functional dependencies \( F \) is redundant.

If there is a \( \ f \in F \), where \( (F - \{f\})^+ = F^+ \), then we say \( F \) is redundant.

Time complexity: \( O(|F|^2) \).

See also: 5.2 Redundancy Tests in Beeri and Bernstein (1979, p. 47)

Parameters
FA set of functional dependencies.

◆ lminimum()

fds fdc::lminimum ( const fds &  F)

Given a set of functional dependencies \( F \), calculate \( G \) where:

  • \( G^+ = F^+ \)
  • \( \forall H^+ = G^+, |H| \geq |G| \).
  • For every \( X \to Y \in G \), there is no such a \( X^{'} \subset X \) where \( X^{'} \to Y \in G \).

Time complexity: \( O(|F|^2) \).

See also: Corollary 2. in Maier(1979, p. 335)

Parameters
FA set of functional dependencies.

◆ non_redundant()

fds fdc::non_redundant ( const fds &  F)

Given a set of functional dependencies \( F \), calculate a non-redundant cover \( G \) of \( F \), where:

  • \( G^+ = F^+ \)
  • \( \forall H \subset G: H^{+} \neq F^+ \)

Time complexity: \( O(|F|^2) \).

See also: 5.2 Redundancy Tests in Beeri and Bernstein (1979, p. 47)

Parameters
FA set of functional dependencies.
