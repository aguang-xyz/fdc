◆ canonical()

fds fdc::canonical ( const int  N,
const fds &  F 
)

Given a set of functional dependencies \( F \), calculate \( G^+ = F^+ \) where

  • \( G \) is non-redundant.
  • For every \( X \to Y \in G \), \( |Y| = 1 \) and there is no such a \( X^{'} \subset X \) where \( X^{'} \to Y \in G \).

Time complexity: \( O(|F|^2) \).

See also: 5.2 Redundancy Tests in Beeri and Bernstein (1979, p. 47)

Parameters
NThe number of attributes.
FA set of cuntional dependencies.

Given a set of functional dependencies \( F \), determine if \( F \) is canonical.

The definition of canonical:

  • \( F \) is non-redundant.
  • For every \( X \to Y \in F \), \( |Y| = 1\) and there is no such a \( X^{'} \subset X \) where \( X^{'} \to Y \in F \).

Time complexity: \( O(|F|^2) \).

See also: 5.2 Redundancy Tests in Beeri and Bernstein (1979, p. 47)

◆ is_lrminimum()

bool fdc::is_lrminimum ( const int  N,
const fds &  F 
)

Given a set of functional dependencies \( F \), determine if:

  • \( \forall G^+ = F^+, |G| \geq |F| \).
  • For every \( X \to Y \in F \), there is no such a \( X^{'} \subset X \) where \( X^{'} \to Y \in F \).
  • For every \( X \to Y \in F \), there is no such a \( Y^{'} \subset Y \) where \( (F - \{X \to Y\} + \{X \to Y^{'}\})^+ = F^+ \).

Time complexity: \( O(|F|^2) \).

See also: The definition of LR-minimum in Maier(1979, p. 331)

NThe number of attributes.
FA set of functional dependencies.

Check if a set of functional dependencies \( F \) is redundant.

If there is a \( \ f \in F \), where \( (F - \{f\})^+ = F^+ \), then we say \( F \) is redundant.

Time complexity: \( O(|F|^2) \).
